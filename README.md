# Object-Detection-Yolo-WebRTC
* For setup
  
  Webpage setup
  - npm install
  - npm install axios

  python setup
  - pip install aiortc
  - pip install "fastapi[all]"
  - pip install ultralytics
  OR
  - pip install -r requirement.txt

* For running
  * Server
    - python<version> main.py
  * Client 1 (Python side)
    - python<version> test.py
  * Client 2 (Webpage)
    - npm run dev
